# SECTION 1 — C++ Linkage Wrapping & Math Sanitization (The Macro Bypass)

# Resolves: "declaration of 'sqrtf' has a different language linkage"
# NDK <math.h> provides C++ inline overloads that conflict with the C-linkage of gu.h.
# We temporarily macro-rename the legacy math definitions prior to inclusion to leave 
# the global math namespace untouched for the NDK standard library.
